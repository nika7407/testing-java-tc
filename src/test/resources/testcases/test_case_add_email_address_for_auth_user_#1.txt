Test Priority: tests successful addition of email addresses

Prerequisites:
1. The fine-grained token must have "Email addresses" user permissions (write)
2. User must be authenticated
3. Email addresses should not already be associated with the account

Include request body with emails array containing one or more email addresses Example body: {"emails": ["newemail@example.com", "anotheremail@example.com"]}

Response status code 201 Created and contains array of added email objects

1. Positive case: Status 201 with array of added email objects
